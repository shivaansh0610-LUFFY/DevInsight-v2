# DevInsight v2: The Zero Bugs Club Dashboard

DevInsight is an elite engineering analytics platform designed to rip past the fluff of a standard GitHub profile and reveal raw engineering horsepower.

## 🚀 Hero Features

- **Sentient Terminal Loader**: An immersive hacker-style terminal that logs real-time semantic processing.
- **AI Processing Core**: Leverages Gemini 2.5 Flash via OpenRouter to evaluate developer maturity and authenticity.
- **Real-Time Data Matrix**: Switches from simulated mockups to live GitHub GraphQL queries for 100% accurate contribution heatmaps.
- **High-Performance Motion**: Physics-based animations powered by `framer-motion` for a premium, spring-loaded dashboard experience.
- **Advanced Glassmorphism**: A smoky-black translucent HUD with interactive cursor spotlight effects.

## 🛠️ Stack

- **Framework**: Next.js 14 (App Router)
- **Analytics**: GitHub GraphQL API v4
- **Intelligence**: OpenRouter / Google Gemini
- **Visuals**: Recharts & Framer Motion
- **Styling**: Pure CSS Modules with Glassmorphism

## 📦 Getting Started

1. **Clone and Install**:
```bash
npm install
```

2. **Set up Environment**:
Create a `.env.local` file and add your keys:
```env
OPENROUTER_API_KEY=your_key_here
GITHUB_TOKEN=your_personal_access_token_here
```

3. **Launch Engine**:
```bash
npm run dev
```
