## 🚀 The Core Engines

The platform is built on 6 specialized analytical engines that process a developer's digital soul:

1.  **Authenticity Engine**: Uses Gemini 2.5 Flash to detect semantic patterns in commits, distinguishing between real engineering and AI-patterned curation.
2.  **Engineering Momentum**: A weekly velocity tracker that reveals productivity "flow" states through a high-performance Area Chart.
3.  **Consistency Engine**: A real-time 365-day contribution topology powered directly by the **GitHub GraphQL API (v4)**.
4.  **Technical Depth**: A multi-axis Radar Chart visualizing holistic systems architecture and language spread.
5.  **Growth Trajectory**: A dynamic bento engine tracking stack expansion and domain maturity.
6.  **Stack Distribution**: A glassmorphism-weighted visualization of a developer's technology core.

## 🛠️ The Technology Stack

-   **Frontend**: Next.js 14 (App Router) with React 18.

-   **Intelligence AI**: OpenRouter (Gemini 2.5 Flash) for behavioral semantic mapping.
-   **Visuals**: Recharts for high-fidelity data visualization.
-   **Motion**: Framer Motion for spring-loaded, physics-based UI.
-   **Branding**: Pure Vanilla CSS Modules with custom Glassmorphism tokens.

## 📦 Getting Started

### 1. Requirements
Ensure you have Node.js 18+ installed.

### 2. Installation
```bash
git clone https://github.com/shivaansh0610-LUFFY/DevInsight-v2.git
cd DevInsight-v2
npm install
```

### 3. Environment Configuration
Create a `.env.local` file in the root directory:
```env
OPENROUTER_API_KEY=your_key_here
GITHUB_TOKEN=your_personal_access_token_here
```
> [!NOTE]
> `GITHUB_TOKEN` is required for the real-time GraphQL heatmap and repository stats.

### 4. Boot the Engine
```bash
npm run dev
```
Open `http://localhost:3000` to enter the matrix.
